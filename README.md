# Bus, Metro, Rail, Ferry — One City, Every Route
A structured, multi-modal transit dataset covering Greater Kolkata's bus, Metro and suburban rail networks.

Kolkata's public transport is genuinely multi-modal — many journeys across the city require combining a bus with the Metro, a train with a short walk between stops that aren't obviously connected. Unlike cities with a single unified transit authority and open API (like London's TfL), Kolkata's transit data is scattered across multiple operators and rarely available in one structured format.

**This solution tries brings together route, stop, and schedule information across:**
- Kolkata Metro (all operational lines)
- Bus routes (state-run and select private routes)
- Suburban rail (Sealdah and Howrah divisions)

**Potential uses:**
- Building a multi-modal journey planner (origin → destination across modes)
- Graph/network analysis of Kolkata's transit connectivity
- Identifying underserved areas or transit deserts
- Transit accessibility and urban mobility research

*This solution uses an early/evolving dataset — bus route coverage in particular is incomplete since no single public source documents Kolkata's private bus network comprehensively.*
